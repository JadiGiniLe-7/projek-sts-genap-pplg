Smart Ronda Scheduler

Selamat datang di web Smart Ronda Scheduler!

Panduan

Web ini adalah sebuah web untuk membuat jadwal ronda otomatis. Web ini dapat memudahkan dalam pembuatan jadwal ronda di kampung-kampung ataupun di komplek perumahan yang belum mempunyai sistem keamanan ronda keliling. Berikut adalah panduan cara menggunakan web ini.

Cara membuat jadwal ronda

1. Isi nama warga yang akan ditugaskan di kolom Input Warga.
2. Setelah mengisi nama, klik tombol Tambah Warga.
3. Lalu, klik tombol Buat Jadwal 1 Bulan (jumlah warga minimal harus 2 agar bisa membuat jadwal).
4. Selamat! jadwal ronda sudah dibuat untuk 1 bulan kedepan, cukup scroll ke bawah untuk melihat jadwal lengkapnya.

Cara mengedit warga

Mengubah nama warga

Jika ingin mengubah nama warga, klik tombol Edit, setelah itu ubah nama di kolom input. Jika sudah selesai, jangan lupa klik tombol Update untuk memperbarui nama warga.

Menghapus warga

Untuk menghapus warga, klik tombol Hapus di kolom aksi.

Cara melihat petugas hari ini

Untuk melihat petugas hari ini, cukup klik tombol Petugas Hari Ini, maka akan ada pemberitahuan tentang siapa yang bertugas hari ini.

Cara mereset jadwal

Untuk mereset jadwal, klik tombol Reset Jadwal.
